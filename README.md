# Snake & Ladders Game

A classic Snake and Ladders game implementation using Python and Pygame.

## Features

- Two-player gameplay
- Randomly generated snakes and ladders
- Interactive dice rolling
- Visual board with numbered cells
- Winner announcement

## Requirements

- Python 3.13
- Pygame library

## Installation

1. Install Pygame:
```bash
pip install pygame
```

## How to Play

1. Run the game:
```bash
python "S&L source file.py"
```

2. Press **SPACE** to roll the dice
3. Players take turns automatically
4. Land on a ladder to climb up
5. Land on a snake to slide down
6. First player to reach position 100 wins!
7. Press **ESC** to exit after the game ends

## Game Controls

- **SPACE**: Roll dice
- **ESC**: Exit game (when game is over)

## Game Rules

- Players start at position 1
- Roll dice to move forward
- If you land on a ladder, climb up to the top
- If you land on a snake, slide down to the bottom
- First to reach position 100 wins!
